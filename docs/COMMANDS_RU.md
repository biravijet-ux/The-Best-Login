# Команды The Best Login

## Основная команда

- `/thebestlogin` — открыть основное окно авторизации

## Игровые команды

- `/login`, `/log` — открыть окно входа
- `/register`, `/reg` — открыть окно регистрации
- `/changepassword`, `/cp` — открыть окно смены пароля

## Чат-режим

Чат-режим нужен как запасной вариант, если требуется ручной ввод через команды.

- `/thebestloginchat login <password>`
- `/thebestloginchat log <password>`
- `/thebestloginchat register <password> <confirmation>`
- `/thebestloginchat reg <password> <confirmation>`
- `/thebestloginchat changepassword <old_password> <new_password> <confirmation>`
- `/thebestloginchat cp <old_password> <new_password> <confirmation>`

## Админ-команды

- `/thebestlogin unregister <nickname>` — удалить регистрацию игрока
- `/thebestlogin changepasswordnick <nickname> <password>` — сменить пароль игроку
- `/thebestlogin reload` — перезагрузить конфиг и серверное хранилище

## Подсказки команд

Для `unregister` и `changepasswordnick` доступны автоподсказки по зарегистрированным никам.
