# Установка The Best Login

## Требования

- Minecraft Forge `1.20.1`
- Forge `47.4.11`
- мод должен быть установлен и на клиенте, и на сервере
- для сборки и серверного окружения рекомендуется Java `17`

## Установка на сервер

1. Поместите `thebestlogin-1.0.0.jar` в папку `mods`.
2. Запустите сервер один раз.
3. Дождитесь создания конфигов в `config/thebestlogin`.
4. При необходимости отредактируйте `config.json`.
5. Перезапустите сервер.

## Установка на клиент

1. Поместите тот же `jar` в папку `mods` клиента.
2. Запустите игру с Forge `1.20.1`.
3. Подключитесь к серверу с установленным The Best Login.

## Важно

- без клиентского мода вход на сервер с The Best Login недоступен
- перед обновлением желательно сделать резервную копию `config/thebestlogin`
