# The Best Login

The Best Login — это мод авторизации для Minecraft Forge 1.20.1, который делает вход на сервер аккуратным, понятным и удобным для обычных игроков.

Мод убирает лишнюю путаницу вокруг регистрации и входа, поддерживает автологин, не заставляет светить пароль в обычном чате и оставляет ключевой контроль на стороне сервера. В результате игрок получает понятный сценарий входа, а администратор — строгую и предсказуемую систему авторизации без лишнего шума.

## Возможности

- удобный экран входа, регистрации и смены пароля
- обязательная установка мода и на клиент, и на сервер
- автологин через локальное клиентское хранилище
- серверная блокировка действий до подтверждения входа
- отдельный чат-режим как запасной сценарий
- админ-команды для сброса регистрации и смены пароля
- хранение данных в JSON-файлах
- локализация на 30 языков

## Поддерживаемая версия

- Minecraft `1.20.1`
- Forge `47.4.11`

## Основные команды

- `/thebestlogin`
- `/login`, `/log`
- `/register`, `/reg`
- `/changepassword`, `/cp`
- `/thebestloginchat ...`
- `/thebestlogin unregister <nickname>`
- `/thebestlogin changepasswordnick <nickname> <password>`
- `/thebestlogin reload`

## Где лежат данные

- серверный конфиг: `config/thebestlogin/config.json`
- серверное хранилище: `config/thebestlogin/playerspw.json`
- клиентское локальное хранилище: `config/thebestlogin/passwords.json`
