# Конфиг и файлы The Best Login

## Серверные файлы

### `config/thebestlogin/config.json`

Основной конфиг мода. Используется для настроек авторизации, таймаутов и поведения сервера.

### `config/thebestlogin/playerspw.json`

Серверное хранилище зарегистрированных игроков.

В этом файле находятся:

- хэш пароля
- соль
- версия схемы хэширования
- служебные данные по последнему входу

## Клиентский файл

### `config/thebestlogin/passwords.json`

Локальное клиентское хранилище для автологина.

## Рекомендации

- не публикуйте содержимое `playerspw.json`
- не редактируйте хранилище вручную без необходимости
- перед обновлением мода делайте резервную копию `config/thebestlogin`
