# Безопасность The Best Login

## Общий подход

The Best Login делает сервер авторитетным в вопросах авторизации игрока.

Это означает:

- сервер сам ограничивает действия игрока до входа
- обычной модификации клиента недостаточно, чтобы обойти ограничения
- данные авторизации передаются через сетевой протокол мода, а не через обычный чат

## Что реализовано

- отдельный и удобный ввод пароля
- обязательный клиентский мод для подключения к серверу с The Best Login
- локальное клиентское хранилище для автологина
- серверное хранение хэшей паролей
- ограничение действий игрока до авторизации

## Практические рекомендации

- используйте мод только из доверенного источника
- обновляйте клиент и сервер одновременно
- не распространяйте серверные файлы с данными авторизации
